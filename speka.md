# Спецификация проекта: PDF Image Injector

## 1. Основная цель
Создать легкое веб-приложение (SPA), которое позволяет пользователю загрузить PDF-файл, добавить на страницы изображения (подпись, логотип, фото) и сохранить измененный PDF. 

**Ключевые ограничения:**
- **Редактирование текста не требуется.**
- Вся обработка должна происходить **на стороне клиента** (в браузере), без загрузки файлов на сервер.
- Максимальная производительность и приватность.

## 2. Стек технологий
| Компонент | Технология | Примечание |
|-----------|------------|------------|
| **Framework** | React (Vite) | Быстрая сборка, современная экосистема |
| **Styling** | Tailwind CSS | Утилитарные классы для скорости верстки |
| **PDF Rendering** | `pdfjs-dist` | Только для отображения (Canvas) |
| **PDF Manipulation** | `pdf-lib` | Для внедрения изображений и сохранения |
| **State** | Zustand / Context | Управление состоянием изображений и страниц |
| **Icons** | Lucide React | Легковесные иконки |

## 3. Пользовательский сценарий (User Flow)
1.  **Старт:** Пользователь видит область для загрузки (Drag & Drop) или кнопку "Загрузить PDF".
2.  **Просмотр:** После загрузки PDF отображается в центре экрана. 
    - Есть панель навигации (страницы влево/вправо).
    - Есть контроллер зума.
3.  **Добавление:** Пользователь нажимает "Добавить изображение", выбирает файл (PNG/JPG).
4.  **Размещение:** Изображение появляется на текущей странице. 
    - Пользователь может перетаскивать его (Drag & Drop).
    - Пользователь может менять размер (Resize handles).
5.  **Сохранение:** Пользователь нажимает "Скачать PDF". 
    - Приложение генерирует новый файл с внедренными изображениями.
    - Браузер предлагает скачать результат.

## 4. Функциональные требования

### 4.1. Загрузка и отображение
- [ ] Поддержка drag & drop PDF файла в окно браузера.
- [ ] Рендеринг текущей страницы PDF в `<canvas>`.
- [ ] Отображение номера страницы и общего количества страниц (например: `1 / 10`).
- [ ] Зум (диапазон 50% - 200%).
- [ ] Адаптация канваса под ширину контейнера.

### 4.2. Работа с изображениями
- [ ] Загрузка изображений форматов PNG и JPG.
- [ ] Каждое изображение — объект в состоянии:
  ```typescript
  interface ImageAnnotation {
    id: string;
    pageIndex: number; // 0-based index
    x: number;         // Координата относительно страницы
    y: number;
    width: number;
    height: number;
    data: ArrayBuffer; // Данные изображения
  }
